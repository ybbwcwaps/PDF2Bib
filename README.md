# PDF2Bib
从PDF中解析出所有的参考文献，显示出来，并下载所有的文献的 BIB 文件

```
.   
├── README.md
├── Code
│   ├── background_rc.py
│   ├── main.py
│   ├── Ui_interface.py
│   ├── utils.py
├── resources
│   ├── background.jpg
│   ├── background.qrc
|	├── icon.ico
├── example
│   ├── *.pdf	# some pdf for test
├── run.bat
├── requirements.txt
```

运行，需要的python库在[requirements.txt](./requirements.txt)中，并注意环境变量配置。

双击`run.bat`即可运行。
